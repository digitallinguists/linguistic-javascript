How to use the Wordlist Editor

In order to use the wordlist editor, you will need a valid input file. This file should be structured as a csv file with the first line indicating the names of all columns and the first column pointing to a unique ID for each row of the file.

Key Bindings

  • [F1] Toggle this help message.
  • [F2] Toggle the settings.
  • [F4] Toggle all columns / basic columns.
  • [TAB] Go to first cell (insert mode).
  • [ESC] Exit insert mode.
  • [PAGE UP] Go to previous rows.
  • [PAGE DOWN] Go to following rows.

Filtering

The Wordlist Editor allows you to filter the entries in multiple ways. With help of the text fields, you can select that only
  • specific taxa,
  • specific concepts, or
  • specific columns
are displayed. For taxa and concepts, just type in the name of the taxon or concept you want to select and separate multiple taxa and concept by a comma. For the filtering of columns, you may also toggle between the display of the basic columns (default) and all columns by pressing [F4], or you may select the columns within the settings menu (press [F2] to show or close the settings).
